Нужно разработать Telegram-бота с поддержкой ИИ для расшифровки медицинских анализов, подписочной моделью и базовой монетизацией.

Бот должен быть написан на Python с использованием aiogram. В качестве дополнительного backend-слоя можно использовать FastAPI (по необходимости). Для хранения данных использовать PostgreSQL. Для кеша и состояний Redis. Проект должен быть упакован в Docker.

Основная логика:

Пользователь отправляет фото или PDF с медицинскими анализами в Telegram-бот.

Дальше файл обрабатывается следующим пайплайном:

Предобработка изображения через OpenCV:
улучшение контраста
перевод в grayscale
удаление шума
подготовка изображения к OCR
Распознавание текста через PaddleOCR:
извлечение текста из изображения или PDF
преобразование результата в чистый текст
Постобработка текста:
очистка от мусора OCR
структурирование данных (показатель, значение, единицы измерения)
Передача структурированного текста в OpenAI API:
модель должна объяснить результаты анализов простым человеческим языком
выделить возможные отклонения
дать общие рекомендации без медицинских диагнозов

Функционал бота:

стартовое меню (/start)
загрузка файлов (фото / PDF)
обработка анализа через описанный пайплайн
вывод результата в чат
система подписки:
бесплатный лимит на N анализов
платная подписка с расширенным доступом
интеграция платежей через ЮKassa или Stripe
хранение пользователей, подписок и истории анализов в PostgreSQL

Архитектура:

aiogram отвечает только за Telegram интерфейс
отдельный модуль OCR (OpenCV + PaddleOCR)
отдельный модуль обработки текста
отдельный модуль работы с OpenAI
отдельный модуль платежей
отдельный модуль работы с БД

Дополнительно:

использовать dotenv для переменных окружения
логирование через logging
предусмотреть асинхронную обработку запросов
сделать код модульным и расширяемым

Цель проекта:

Создать Telegram-бота, который принимает медицинские анализы, автоматически распознает их, объясняет простым языком и монетизируется через подписку.
